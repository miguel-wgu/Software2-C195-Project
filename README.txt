Title: Appointment Scheduler

Purpose:
The purpose of this Appointment Scheduler is to enable users to create, update and delete customer records and appointments in the database.
It also provides users with the capability to view reports and keep track of user login activity.

Author: Miguel Guzman
Contact Information: mguz123@wgu.edu
Appointments Scheduler Version: 2.0
Date: 3/27/2023

IDE: IntelliJ IDEA 2022.2.3 (Ultimate Edition)
JDK: 17 Oracle OpenJDK version 17.0.5
JavaFX: JavaFX-SDK 17.0.6

Directions:
Begin by running the program.
The login screen will appear.
Input the username "test" and the password "test" or username "admin" and the password "admin".
Once logged in, you can create, update, and delete customer records and appointments, as well as view reports.

Additional Report:
The additional report displays the number of appointments by division and country.

MySQL Connector version and update number: mysql-connector-j-8.0.26
